# Learning Matplotlib
Repository of matplotlib experiments. Purpose of this repository is to become an expert in Matplotlib,
especially for image processing and visualization.

## Environment setup
Download and install latest version of Miniforge3: https://conda-forge.org/download/
Make sure to add the tools to PATH.

Run the following commands in your terminal:
- mamba create -n learningmatplotlib python=3.11
- mamba activate learningmatplotlib
- mamba install -c conda-forge numpy pandas opencv pytest pyinstaller matplotlib
- mamba install -c pytorch -c nvidia pytorch torchvision torchaudio pytorch-cuda=12.1
